== Description ==

= Simple Scroll To Top With Icon =

Simple Scroll To Top With Icon adds a smooth scroll-to-top button to your WordPress site, complete with a customizable icon.

This lightweight plugin is easy to set up. Just activate it and configure the settings from the “Scroll To Top With Icon” menu tab.

Plugin documentation: [https://wordpress.org/plugins/simple-scroll-to-top-with-icon/] (https://wordpress.org/plugins/simple-scroll-to-top-with-icon/)

== Installation ==

1. Install Simple Scroll To Top With Icon by uploading the `simple-scroll-to-top-with-icon` zip file to the `/wp-content/plugins/` directory.
2. Activate Simple Scroll To Top With Icon through the `Plugins` menu in WordPress.

== Frequently Asked Questions ==

= What does this plugin do? =

Simple Scroll To Top With Icon adds a customizable scroll-to-top button to your website, providing a smoother browsing experience.

= Can I customize the icon? =

Yes, you can choose a custom icon from the settings panel.

= How can I change the settings of the scroll button? =

Navigate to “Scroll To Top With Icon” in the WordPress admin panel. Here you’ll find all the settings you need.

= Will Simple Scroll To Top With Icon slow down my website? =

No, the plugin is optimized for performance.
